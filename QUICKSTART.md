# 项目核心流程说明

## 🎯 这个项目是什么？

**AI 语音匹配平台** - 一个通过语音对话进行智能匹配的系统

## 📋 核心功能（按重要性排序）

### 1. 智能对话 (LLM) ⭐⭐⭐ 最核心
- **功能**: 与 AI 进行对话
- **提供商**: Kimi (Moonshot)
- **验证方式**: 发送文字，收到回复

### 2. 用户匹配 (Matcher) ⭐⭐ 核心业务
- **功能**: 根据用户特征匹配相似用户
- **算法**: 多维度相似度计算
- **验证方式**: 提交用户信息，获取匹配结果

### 3. 语音识别 (ASR) ⭐ 辅助功能
- **功能**: 语音转文字
- **提供商**: 科大讯飞
- **验证方式**: 上传音频，返回文字

### 4. 语音合成 (TTS) ⭐ 辅助功能
- **功能**: 文字转语音
- **提供商**: 科大讯飞
- **验证方式**: 输入文字，返回音频

## 🔄 完整使用流程

### 最简单的流程（文字对话 + 匹配）：

```
用户输入文字 → LLM生成回复 → （可选）语音播放
                    ↓
                分析用户特征
                    ↓
                返回匹配结果
```

### 完整流程（语音 + 对话 + 匹配）：

```
用户说话 → ASR识别成文字 → LLM生成回复 → TTS转成语音 → 播放
                                     ↓
                                 分析特征
                                     ↓
                                 返回匹配
```
